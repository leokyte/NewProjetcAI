# API Reference

Referência dos principais módulos de serviço usados pelo app, organizados por domínio.

- Serviços HTTP base: `src/services/kyte-api-gateway.js`
- Conta/Usuário/Loja: `src/services/kyte-account.js`
- Dados (sync, estoque, relatórios): `src/services/kyte-data.js`
- Integrações (TikTok, Facebook Pixel, Payment Link): `src/services/kyte-api.js`
- Web (Produto/Categoria/IA): `src/services/kyte-api-web.ts`
- Backend (Variações/SKUs): `src/services/kyte-backend.ts`
- Autenticação: `src/services/kyte-auth.js`
- E-mail (recibos): `src/services/kyte-sender.js`

Para exemplos detalhados, veja `docs/api/services.md` e `docs/services/KyteServices.md`.
